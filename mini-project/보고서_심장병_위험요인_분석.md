## 1. 개요

- **과제**: Heart Disease UCI 데이터를 이용한 상관관계 분석
- **목표**: 심장병(target)과 가장 관련이 깊은 변수 파악 (예: 콜레스테롤, 최대 심박수 등)
- **데이터**: `data/heart_cleaned.csv` (Heart Disease UCI 기반 정제 데이터)

## 3. 분석 방법

### 3.1 컬럼 선택 이유

- **대상**: target을 제외한 **모든 숫자형 컬럼 13개** 사용  
- **이유**  
  - 심장병(target)과의 관계를 **모든 후보 변수**에 대해 공정하게 비교하기 위함  
  - 상관계수가 계산 가능한 숫자형 변수를 빠짐없이 포함  
  - 시각화에서 **절댓값 정렬**로 “가장 관련이 깊은 변수”가 자동으로 상위에 오도록 구성  

### 3.2 상관분석 방법

- **지표**: 피어슨 상관계수 (Pearson correlation)
- **범위**: -1 ~ +1  
  - 양수: target(심장병)과 같은 방향으로 움직임  
  - 음수: 반대 방향  
  - 절댓값이 클수록 관련성 강함  

### 3.3 시각화: 히트맵 선택 이유

- 상관관계를 한눈에 비교하기 위해 **히트맵** 사용  
- target은 축에 넣지 않고, **「target과의 상관계수」 한 행(1×13)**만 표시  
  - 심장병과 각 변수의 관련 강도(양/음, 크기)를 색으로 비교  
- **정렬**: 절댓값 기준 내림차순 → 관련이 깊은 변수 순으로 x축 배치  
- **도구**: Plotly Express `px.imshow`, 색상 스케일 `RdBu_r`, 범위 -1 ~ 1  
- x축 라벨은 **한글 변수명**으로 표기  

---

## 4. 결과 요약

- **히트맵**에서 x축은 “심장병과의 상관계수 절댓값이 큰 순”으로 정렬된 13개 변수(한글명)이며, y축은 「target과의 상관계수」 한 행입니다.  
- 색이 **진한 빨강**에 가까울수록 심장병과 **양의 상관**, **진한 파랑**에 가까울수록 **음의 상관**을 의미합니다.  
- 이를 통해  
  - 심장병과 **가장 관련이 큰 변수**가 무엇인지,  
  - 콜레스테롤·최대 심박수 등 특정 변수가 상대적으로 얼마나 중요한지  
를 직관적으로 비교할 수 있습니다.  

실제 분석 시에는 `mini_project.py`를 실행하여 생성되는 **히트맵 그래프**를 보고, 상위에 위치한 변수들을 “심장병 위험 요인 후보”로 해석하면 됩니다.
